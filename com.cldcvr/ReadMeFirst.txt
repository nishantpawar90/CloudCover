Execution
To execute the project there are two different ways.
1. Import the project in Eclipse and navigate to src\test\java\runner and run the testng.xml as TestNGSuite.
2. Import the project in Eclipse and navigate to root directory and run the pom.xml by executing command mvn test.


Result
For quicker view of result, the emailable-report.html file is available at root directory. If the suite is executed then the
results can be found out at test-output folder.
